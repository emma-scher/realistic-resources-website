What This Prompt Does

This prompt helps you turn vague, overwhelming, or unclear tasks into specific, actionable next steps. It's designed for moments when you know you need to do something, but the path forward isn't clear.

Best for:

  • Breaking down complex projects when you're feeling overwhelmed
  • Clarifying ambiguous work requests
  • Planning when executive function is low
  • Getting unstuck when you don't know where to start

How to Use This Prompt

  1. Copy the prompt text from the box below
  2. Paste it into your AI tool (Claude, ChatGPT, etc.)
  3. Replace the bracketed sections with your specific information
  4. Review the output and adjust as needed for your situation

The Prompt

I need help breaking down a task into actionable steps. Task: [Describe your task or project here] Current situation: - What I know: [What information you already have] - What I don't know: [Gaps in information or understanding] - Constraints: [Time limits, resources, energy levels, etc.] - Desired outcome: [What success looks like] Please help me: 1. Identify the core components of this task 2. Break it into specific, sequential steps 3. Flag any dependencies or things I need before starting 4. Suggest realistic time estimates for each step 5. Highlight potential obstacles and how to prepare for them Focus on making this practical and achievable given my constraints.

Example Use Case

Original task: "Prepare for quarterly review meeting"

After using this prompt:

  1. Gather Q4 project data (30 min)
  2. Review previous quarter's goals (15 min)
  3. Draft 3-5 key accomplishments (20 min)
  4. Identify 2-3 challenges and solutions (15 min)
  5. Prepare questions for manager (10 min)
  6. Review and refine notes day before meeting (15 min)

When This Prompt Is Useful

  • You're staring at a task and don't know where to begin
  • The scope feels overwhelming or too big to tackle
  • You have limited energy and need to be strategic
  • The request is vague or poorly defined
  • You need to estimate time or plan around other commitments

Customizing This Prompt

You can adapt this prompt by:

  • Adding specific formatting preferences (e.g., "organize as a checklist")
  • Requesting priority ordering ("rank steps by importance")
  • Including accessibility needs ("I need verbal instructions, not visual")
  • Specifying your working style ("I work better with small, frequent tasks")
